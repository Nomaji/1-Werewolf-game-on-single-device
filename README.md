# 🐺 本格対面人狼 Web App (GM-less Edition)

1台のデバイスを回しながら、GM（進行役）なしで遊べる本格的な対面人狼支援ツールです。
基本的な役職に加え、**「純愛者」「サイコ」「わら人形」**などの特殊役職を多数搭載し、複雑な裁定もシステムが自動処理します。

## 🚀 特徴

- **完全GMレス**: アプリが進行をナビゲートするため、全員がプレイヤーとして参加可能。
- **高度な役職ロジック**: 
  - 純愛者の身代わり・後追い処理。
  - サイコによる夜のアクション返り討ち。
  - わら人形の道連れ選択シークエンス。
- **議論サポート**: 昼の議論中に各プレイヤーの「CO（役職公表）」や「メモ」を個別に記録可能。
- **クイックセットアップ**: 6人/9人/13人の推奨編成プリセットを完備。
- **レスポンシブ設計**: スマートフォンでの操作に最適化。

## 🎮 遊び方

1. **Setup**: プレイヤー名を入力し、役職の配分を決定します。
2. **Role Confirm**: 1人ずつ端末を受け取り、自分の役職を確認します（双子や人狼、スパイには固有の情報が表示されます）。
3. **Day Phase**: タイマーに従って議論します。役職COボタンやメモを活用してください。
4. **Voting**: 処刑するプレイヤーを選びます（一斉投票・個別投票に対応）。
5. **Night Phase**: 役職に応じた行動（占い、襲撃、暗殺、護衛など）を順番に30秒以内に行います。
6. **Result**: 勝利条件を満たすと結果が表示されます。終了後は名前を保持したまま再戦可能です。
